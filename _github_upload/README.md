# 医学古籍诗云 5.0

采用 `Vibe-coding--poetry-galaxy-main.zip` 中的 `5.0` 版本形态：Three.js + OrbitControls 的 3D 粒子星云。

## 打开方式

```text
http://127.0.0.1:5178/index.html
```

## 当前形态

- 3D 粒子星云，按古籍来源聚类
- 鼠标拖拽旋转宇宙
- 滚轮缩放
- 每个可交互粒子对应一条医学古籍语料
- 点击粒子弹出古籍详情
- 每本古籍生成太阳级主星
- 高频医理/病机主题生成富集亮星
- 普通语料作为星尘围绕所属古籍主星分布
- 背景 dust 粒子增强星河纵深

## 古籍数据

当前使用医学古籍种子句段扩展为 `10000` 条语料，覆盖：

《黄帝内经》《难经》《伤寒论》《金匮要略》《神农本草经》《千金方》《外台秘要》《和剂局方》《脾胃论》《丹溪心法》《景岳全书》《本草纲目》《温热论》《湿热病篇》《温病条辨》《临证指南医案》等。
